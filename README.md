# API de Gerenciamento de Tarefas

Projeto desenvolvido após a graduação em Análise e Desenvolvimento de Sistemas, com o objetivo de aplicar boas práticas de desenvolvimento back-end.

## Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Bean Validation

## Funcionalidades
- Cadastro de tarefas
- Listagem de tarefas
- Marcação de tarefa como concluída

## Arquitetura
- Controller
- Service
- Repository
- DTOs

## Documentação da API

A documentação da API pode ser acessada via Swagger:
http://localhost:8080/swagger-ui.html
